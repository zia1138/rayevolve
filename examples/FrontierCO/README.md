
# FrontierCO

Examples from the "FrontierCO: A Comprehensive Evaluation of Contemporary ML-Based Solvers for Combinatorial Optimization"
paper.

https://github.com/sunnweiwei/FrontierCO
https://arxiv.org/pdf/2505.16952
https://huggingface.co/datasets/CO-Bench/FrontierCO


To download the "training" instancees from huggingface hub, run the following command:
```bash
python dl_data.py
```

To download both the "training" and "test" instancees from huggingface hub, 
run the following command:
```bash
python dl_data.py --all
```
